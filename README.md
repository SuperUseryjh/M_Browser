# M_browser 项目文档

## 1. 项目简介

M_browser 是一个基于 Uni-app 框架开发的移动端浏览器应用。它旨在提供一个简洁、高效的浏览体验，并支持多平台发布。

## 2. 主要功能

- **网页浏览:** 支持基本的网页加载和显示。
- **搜索功能:** 集成搜索页面，方便用户进行内容搜索。
- **底部导航栏:** 提供便捷的底部导航，方便用户在不同页面间切换。
- **状态栏适配:** 针对不同设备的状态栏高度进行适配。
